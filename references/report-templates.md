# 报告架构标准 Report Architecture Standards
## 7种报告模板 + 排版与叙事规范

---

## 模板选择指南

| 模板 | 适用场景 | 页数 | 核心受众 |
|------|----------|------|----------|
| T1 行业速报 | 快速了解新行业 | 4-6页 | 决策者/投资人 |
| T2 标准行业研报 | 系统性行业认知 | 20-35页 | 分析师/战略团队 |
| T3 深度投资研究 | 投资决策支持 | 35-55页 | 投资委员会/基金经理 |
| T4 竞争对手情报 | 竞争战略制定 | 15-25页 | 战略/BD团队 |
| T5 市场进入评估 | 新市场决策 | 25-40页 | 管理层/创始人 |
| T6 技术颠覆报告 | 技术战略规划 | 20-30页 | CTO/技术战略 |
| T7 监管风险简报 | 合规/政策决策 | 10-15页 | 法务/政府关系 |

---

## T1：行业速报（Executive Flash）

```
封面
├── 行业名称 + 副标题（一句话定义这个行业）
├── 研究日期 + 数据截止时间
└── 3个核心判断（加粗，放封面）

1. 30秒摘要（≤200字）
   核心数字 + 最重要的3个判断 + 1个行动建议

2. 市场快照
   ├── 规模：XXX亿元/美元（CAGR XX%，2024-2029）
   ├── 主要玩家：Top3 + 份额
   ├── 阶段：[萌芽/高增/成熟/整合/衰退]
   └── 关键驱动/阻力（各3条，用数据支撑）

3. 竞争格局一图
   ├── 市场份额饼图 or 战略定位矩阵
   └── 1-2段文字解读关键格局特征

4. 核心观点（≤3个）
   每个观点：1行结论 + 3行支撑 + 1行行动含义

5. 需要进一步研究的问题（≤5个）
```

## T2：标准行业研报

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
封面页
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
│ [行业名称]深度研究报告
│ 副标题：一句话点明核心论点
│ 研究机构 | 日期 | 版本
│ 
│ ▶ 核心结论（3-5条，直接给答案）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
目录
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

一、执行摘要 (2-3页)
├── 本报告研究的核心问题
├── 主要发现（结论先行）
└── 关键数据速查表

二、行业全景 (3-5页)
├── 2.1 行业定义与边界
├── 2.2 市场规模与增速
├── 2.3 产业链图谱（上中下游）
├── 2.4 市场分层矩阵
└── 2.5 行业生命周期定位

三、竞争格局 (4-6页)
├── 3.1 整体格局快照（市场集中度）
├── 3.2 主要玩家画像（Top 5-8）
│   ├── 战略定位
│   ├── 核心财务指标
│   └── 竞争优势与弱点
├── 3.3 战略群组分析
└── 3.4 新进入者与潜在颠覆者

四、技术与创新 (2-3页)
├── 4.1 关键技术图谱
├── 4.2 颠覆向量评估
└── 4.3 AI渗透现状与路径

五、财务与估值 (3-4页)
├── 5.1 商业模式对比
├── 5.2 利润结构与行业毛利率分布
├── 5.3 资本密度与ROIC水平
└── 5.4 估值倍数参考范围

六、监管与政策 (2-3页)
├── 6.1 监管框架概览
├── 6.2 重要政策节点
└── 6.3 未来政策变化扫描

七、消费者与需求 (2-3页)
├── 7.1 核心用户画像
├── 7.2 需求趋势变化
└── 7.3 痛点与市场缺口

八、核心洞察与战略含义 (3-4页)
├── 8.1 三大结构性判断（Insight-A级）
├── 8.2 情景分析（2×2矩阵）
├── 8.3 风险矩阵（影响×概率）
└── 8.4 行动建议（分受众类型）

附录
├── 数据来源清单（分级标注）
├── 数据空洞记录
└── 方法论说明
```

## T3：深度投资研究报告

```
在 T2 基础上，增加以下模块：

投资摘要（1页）
├── 投资评级（超配/中性/低配 + 目标价/目标倍数）
├── 核心投资逻辑（3条，每条≤2行）
├── 主要风险（3条）
└── 触发信号（什么事件会改变判断）

财务模型补充
├── 收入预测（3年期，分驱动因子拆解）
├── 利润率假设说明
├── DCF主要参数及敏感性
└── 可比公司估值表（EV/EBITDA P/E P/S）

行业深度分析补充
├── 历史周期分析（过去2-3个完整周期复盘）
├── 护城河深度评分（参考Morningstar框架）
├── 管理层质量评估
└── ESG风险打分

对立观点章节（Anti-thesis）
├── 空头逻辑是什么？
├── 做空催化剂有哪些？
└── 我们为什么认为空头是错的（或部分对的）
```

## T4：竞争对手情报报告

```
一、情报摘要（1页）
└── 最重要的3个竞争信号 + 建议行动

二、目标公司档案（每家2-3页）
├── 基本面：规模/增速/盈利/资金状况
├── 战略重点：当前押注在哪里
├── 执行进展：交付情况 vs 承诺
├── 弱点与脆弱点
└── 可能的下一步动作

三、相对竞争力评估
├── 维度比较矩阵（10个维度打分）
├── 我方vs竞争对手的胜负面分析
└── 未来12个月的竞争态势预判

四、行动建议
├── 立即行动：防御/进攻/围堵
├── 6个月行动：战略调整项
└── 需要持续监测的指标
```

## T5：市场进入评估报告

```
一、执行结论（1页）
└── 建议进入/不进入 + 如果进入，怎么进

二、市场机会评估
├── 市场规模与增速
├── 可进入的细分赛道（避开强手的机会窗口）
└── 进入时机评估（是否是最佳时间？）

三、竞争壁垒评估
├── 现有玩家防御力评分
├── 我方进入的差异化路径
└── 最小有效规模（MES）要求

四、进入路径选项
├── 选项A：有机增长（自建）
│   成本/周期/成功率/关键假设
├── 选项B：并购进入
│   标的清单/估值区间/整合难度
└── 选项C：合作/授权
    合作对象/条件/风险

五、财务预测
├── 情景一（基准）：投入/收入/回报
├── 情景二（乐观）：关键假设变化影响
└── 情景三（悲观）：损失上限与退出机制

六、进入计划与关键里程碑
└── 18个月行动路线图
```

## T6：技术颠覆报告

```
一、颠覆信号摘要
└── 正在发生的3个颠覆信号 + 紧迫性评估

二、当前技术格局
├── 主流技术体系图
├── 成熟度评估（Gartner曲线定位）
└── 主要技术持有者及护城河

三、颠覆向量深度分析
├── 颠覆技术1：成熟度/商业化时间/颠覆范围
├── 颠覆技术2：成熟度/商业化时间/颠覆范围
└── 综合颠覆情景（最终态是什么？）

四、价值链重塑分析
├── 哪些环节会被消除
├── 哪些环节会被增强
└── 新兴环节/新商业模式

五、赢家/输家判断
├── 现有玩家中谁最脆弱
├── 谁有最佳转型条件
└── 新进入者的最大机会窗口

六、建议行动
├── 短期：监测/防御/探索
└── 中期：押注/转型/撤退
```

---

## T7：监管风险简报

```
一、风险等级评估
└── 高/中/低 + 紧迫性时间表

二、当前监管框架
├── 主管机构 + 现行规则
└── 合规状态评估

三、变化预警
├── 已确认变化（已公布草案/征求意见稿）
├── 高概率变化（基于政策信号）
└── 低概率高影响变化（尾部风险）

四、影响评估
├── 对收入的影响（量化）
├── 对成本的影响（量化）
└── 对竞争格局的影响

五、应对建议
├── 需要立即行动的合规事项
├── 政策倡导机会
└── 风险对冲方案
```

## 通用排版与叙事规范

### 数字呈现规范

```
市场规模：优先用亿元（中国）或十亿美元（全球）
增速：精确到小数点后1位，如 12.3% CAGR
市场份额：精确到小数点后1位
利润率：精确到小数点后1位
估值倍数：整数或小数点后1位
```

### 图表使用规范

```
必须包含的图表（每份报告）：
□ 市场规模趋势图（柱状/折线）
□ 市场份额图（饼图或条形图）
□ 价值链图（含各环节利润率）
□ 竞争格局矩阵（2×2定位图）
□ 情景分析矩阵（2×2）

每张图必须：
□ 标注数据来源
□ 标注数据时间
□ 有1-2行文字解读（不能只有图没有解读）
```

### 标题层级规范

```
报告大标题：表达核心判断，不做描述性标题
✅ "平台壁垒正在软化：外卖行业的格局重塑时刻"
❌ "2024年中国外卖行业分析报告"

章节标题：结论式标题
✅ "技术壁垒是真实的，但窗口期在缩短"
❌ "技术分析"

小节标题：问题式或结论式
✅ "谁在控制定价权？" 或 "头部集中度仍在加速"
❌ "定价情况分析"
```

### 摘要写作规范

```
执行摘要的三段式结构：
第1段：核心结论（先说最重要的判断）
第2段：支撑逻辑（2-3个关键事实/数据）
第3段：行动含义（对谁意味着什么，该怎么做）

长度：整份报告的 5-8%（20页报告，摘要 1-1.5页）
```
